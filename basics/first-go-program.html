First Go Program

How to write a Go program

  • Create directory inside GoPath: mkdir -p src/github.com/name
  • Create file hello.go with the content:
package main

import "fmt"

// this is a comment
func main() {
    fmt.Println("Hello, World")
}
  • Run go run main.go and the output on terminal is: $ Hello World

How to read a Go program

  • the program is readed from top to bottom, left to right
  • package main is the package declaration. There are two types of go programs: executables and libraries. Every executable - program - needs a package main. Libraries are a way of reusing code and for convention they get the same package name as the folder.
  • the white line has no meaning for the compiler. They are usefull only to make the code more readable.
  • import fmt is how we include code from other packages.
  • // represents a comment. This is usefull for make the code more readable and for go to generate documentation.
  • func main() {} is a function that represents the execution point of the program. Every executable should have a main() function.
  • fmt.Println("Hello, World") is the syntax to access a function Println() from the package fmt. The function tekes one string argument Hello World and prints with new line the string to the terminal.

Find more information

Type into the terminal: godoc fmt Println
