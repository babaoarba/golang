Data types

Golang Data Types

The information, in a computer, is stored into capacitors and transistor as electric impulses. Lower impulse than 0.8V represent a zero and a higher impulse, up to 5V, represent an one. The zero and one, in programming, are called bits.

Computers operate on fixed sized bits (numbers) called words. Words can be interpreted as integers, floating point numbers, bit sets or memory address that can be combined into larger aggregated that represent pixels, books, ecc.

Go offers four ways to orgagnise data:

  • basic types: numbers (integers, floating point numbers, complex numbers), booleans, strings
  • aggregate types: arrays and structs
  • reference types: slices, maps, functions, pointers, channels
  • interface type

Go binary operators

Go binaries operators for logic, comparison and arithmetic can be applied on types and are: 

* / % << >> & &^ +-|^
== != < <= > >=
&&
||

The operators are in decreasing order.

Overflow

Overflow in Golang is safe.

var u uint8 = 255
fmt.Println(u, u+1, u*u) // "255 0 1"
var i int8 = 127
fmt.Println(i, i+1, i*i) // "127 -128 1"
