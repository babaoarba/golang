Basic data types

Basic Data Types

Integers

There are five sizes if integers:

  • int8, int16, int32, int64, int with corresponding unsigned versions uint8, uint16, uint32, uint64, uint. The int data type depend on the architecture of the computer and is assigned automatically by the compiler.

There are also two synonyms:

  • rune that is synonym of int32 and indicates that the value is a Unicode code point.
  • byte that is synonym of int8 and indicates that the value is a pice of raw code.

Finally there is uintptr that can hold the bits of any pointer value.

var money int = 0

Floating Point Numbers

Go provides two sizes of floating point numbers: float32 and floaat64. A float32 provides a six decimal digits precision. The float64 provides almost 15 digits.

var f float = 1.2

Booleans

Booleans has only two possible values: true and false. Boolean values can be combined with the && (AND) and || (OR) operators, which have short-circuit behavior : if the answer is already determined by the value of the left operand, the right operand is not evaluated, making it safe to write expressions like this:

s != "" && s[0] == 'x'

where s[0] would panic if applied to an empty string.

Strings

A string is an immutable sequence of bytes. Text strings are conventionally interpreted as UTF-8-encoded sequences of Unicode code points (runes).

s := "hello, world"     // "hello world" represent a string literal
fmt.Println(len(s))     // "12"
fmt.Println(s[0], s[7]) // "104 119"  ('h' and 'w')

Since strings are immutable and constructors like this are invalid:

s[0] = 'L' // compile error: cannot assign to s[0]

Immutability means that it is safe for two copies of string to share the same underlying memory, making it cheap to copy strings of any length. Similarly, a string s and a substring s[7:] may safely share the same data; no new memory is allocated in either case.

String literals

"Hello World" represent a string literal. Inside double quoted string literal can be inserted ASCII control lines:

\a alert or bell
\b backspace
\f form feed
\n new line
\r carrige return
\t tab
\v vertical tab
\' single quote (only inside rune literal '\'')
\" double quote (only within " " literals)

Arbitrary bytes can also be included in literal strings using hexadecimal or octal escapes.

Raw literals

A raw string literal is written ..., using backquotes instead of double quotes. Within a raw string literal, no escape sequences are processed; the contents are taken literally, including backslashes and newlines, so a raw string literal may spread over several lines in the program source.

UTF-8

Go source files are always encoded in UTF-8, and UTF-8 is the preferred encoding for text strings manipulated by Go programs. There are two form of unicode: \uhhhh for a 16-bit value and \Uhhhhhhhh for a 32-bit value, where each h is a hexadecimal digit

Usefull packages

Four standard packages are particularly important for manipulating strings: bytes, strings, strconv, and unicode.

The strings package provides many functions for searching, replacing, comparing, trimming, splitting, and joining strings.

The bytes package has similar functions for manipulating slices of bytes, of type []byte, which share some properties with strings. Because strings are immutable, building up strings incrementally can involve a lot of allocation and copying . In such cases, it’s more efficient to use the bytes.

func intsToString(values []int) string {
    var buf bytes.Buffer
    buf.WriteByte('[')
    for i, v := range values {
        if i > 0 {
            buf.WriteString(", ")
        }
        fmt.Fprintf(&buf, "%d", v)
    }
    buf.WriteByte(']')
    return buf.String()
}

func main() {
    fmt.Println(intsToString([]int{1, 2, 3})) // "[1, 2, 3]"
}

The strconv package provides functions for converting boolean, integer, and floating-point values to and from their string representations, and functions for quoting and unquoting strings.

The unicode package provides functions like IsDigit, IsLetter, IsUpper, and IsLower for classifying runes. Each function takes a single rune argument and returns a boolean. Conversion functions like ToUpper and ToLower convert a rune into the given case if it is a letter. All these functions use the Unicode standard categories for letters, digits, and so on. The strings package has similar functions, also called ToUpper and ToLower, that return a new string with the specified transformation applied to each character of the original string.

Constants

Constants are expressions whose values are known at compile time. Constants are made by basic types: booleans, strings or numbers.

const pi = 3.14159 // approximately; math.Pi is a better approximation

To create enumerators, in Go, can be used the iota keyword.

type Weekday int

const (
    Sunday Weekday = iota
    Monday
    Tuesday
    Wednesday
    Thursday
    Friday
    Saturday
)

This declares Sunday to be 0, Monday to be 1, and so on.
