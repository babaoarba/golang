Structure

Names

Part of code that is written is composed by custom names for variables, functions, ecc. The names need to begin with a letter or undescore and can have any number of additional letters, undescores, digits. The names should be camel case, should be short for local scope and more meaningfull and longer for larger scope.

Keywords

Go has 25 keywords:

break, default, func, interface, select, case, defer, go, map, struct, chan, else, goto, package, switch, const, fallthrough, if, range, type, continue, for, import, var, return

Scope

Is a part of code at wich a declared name refers to.

...

func f() {}

var g = "g"

func main() {
    f := "f"
    fmt.Println(f) // "f"; local var f shadows package-level func f
    fmt.Println(g) // "g"; package-level var

There are four types of declarations: var, cons, type, func.

Structure

A Go program is stored in one or more files that end in .go. Each file begin with package that says what package the file is part of. After package there is an import declaration and then a sequace of package level declarations.
