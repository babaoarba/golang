Benchmarking

Benchmarking is a way of testing the performance of code. it's useful to test different solutions to the same problem.

func Add(a, b float64) float64 {
     return a + b
}

func Sub(a, b float64) float64 {
    return a - b
}

To test the performance of the above functions we use the testing file or if not created create an new one with the content:

func BenchmarkAdd(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Add(2, 3)
    }
}

func BenchmarkSub(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Sub(2, 3)
    }
}

Running the benchmark

go test -bench=.

Output:

Name Benchmark Number of executions Time
BenchmarkAdd-4 2000000000 0.50 ns/op
BenchmarkASub-4 2000000000 0.49 ns/op

Benchmark commands

go test -bench=BenchmarkSub     // runs only the function specified
go test -bench=. -run=NONE   // skip testing and run all the benchmark
go test -bench=. -benchmem      // show memory allocation
