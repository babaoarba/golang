Testing

Testing

Writing tests for our code is a good way to ensure quality and improve reliability.

Inside math folder from packages create a math_test.go file with the content:

package math

import "testing"

func TestAverage(t *testing.T) {
    v := Average([]float64{1, 2})
    if v != 1.5 {
        t.Error("Expected 1.5, got ", v)
    }
}

go test and the result should be PASS

Commands for testing

go test // test all functions
go test . // like go test
go test -run=TestAverage // tests only the function specified
go test -v // result is verbose
go test -cover // show how much of actual code is getting tested

// Output test result to file
go test -coverprofile=cover.out // save to file cover.out the test result
go tool cover -func=cover.out   // pretty output of the contents of file
go tool cover -html=cover.out   // show result as HTML
