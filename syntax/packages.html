Packages

Packages

Packages is a mechanism that allow the reuse of code.

import "fmt"

fmt is the name of package that includes functions to format and output on the screen.

Packages are useful to:

  • avoid overlapping names and keeping meaningful function names
  • organize code
  • compilation of code is done once.

Create a package

Make a directory inside gopath folder mkdir -p src/Play/math and insert into math.go file:

// math.go

package math

func Average(xs []float64) float64 {
    total := float64(0)
    for _, x := range xs{
        total += x
    }
    return total / float64(len(xs))
}

Use the directory from main:

package main

import (
    "Play/math"
    "fmt"
)

func main() {
    a := math.Average([]float64{1, 3, 4.5})
    fmt.Println(a) // 2.83333
}

go build and run the executable.
