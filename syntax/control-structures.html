Control Structures

Control structures

for loop

A for statment allows us to repeat part of code multiple times. The for loop is the only loop statment in Go and has a number of forms:

for

for initialization; condition; post { 
    // zero or more statements
}

for i := 1; i <= 10; i++ {
    fmt.Println(i)
}
  • the loop prints numbers from 1 to 10
  • i is not visible outside of the loop
  • initialization is executed first. Then the condition is verified and if true executes the statements inside the loop. When program reach the last } of the loop it executes the post and then check the condition again.

while

In Go the while loop is created using for

i := 1
for i <= 10 {
    fmt.Println(i)
    i++
}
  • the loop prints numbers from 1 to 10
  • i is visible outside of the loop and other parts of program can modify the variable

infinite loop

for {
    // run forever
}

The infinite loop can be terminated with break or return.

range loops

func main() {
    vals := []int{4, 2, 6}
    for _, v := range vals {
        v--
        fmt.Println(v)
    }
    fmt.Println(vals)
}

if and else

if statment is useful to take decisions based on a condition.

if name == "Andrei" {
    fmt.Println("Hello Andrei")
}
  • the code runs only if name variable is Andrei
if i%2 == 0 {
    // even
} else {
    // odd
}
  • the condition is checked and if true the if statement are executed otherwise the else will be executed.
if i%2 == 0 {
    // divisible by 2
} else if i%3 == 0 {
    // divisible by 3
} else if i%4 == 0 {
    // divisible by 4
}
  • with the combination of else if multiple consitions can be checked.

switch

switch is similar to else if but is more readable when testing multiple conditions.

switch i {
    case 0:
        fmt.Println("Zero")
    case 1:
        fmt.Println("One")
    case 2:
        fmt.Println("Two")
    case 3:
        fmt.Println("Three")
    case 4:
        fmt.Println("Four")
    case 5:
        fmt.Println("Five")
    default:
        fmt.Println("Unknown Number")
}
