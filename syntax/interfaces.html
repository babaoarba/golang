Interfaces

Interfaces

Interfaces are an abstract type. They describe the behaviour that other types need to have.

type Shape interface {
    area() float64
}

Interfaces are created with interface keyword and contains methods that other types need to implement.

Implement an interface

type Circle struct {
    r float64
}

func (c *Circle) area() float64 {
    return math.Pi * (c.r * c.r)
}

type Rectangle struct {
    l, w float64
}

func (r *Rectangle) area() float64 {
    return r.l * r.w
}

Circle and Rectangle has two methods that implements the Shape interface by using the area method defined by Shape. Now, both Circle and Rectangle are shapes and we can write a generic function that both Circle and Rectangle can use:

func totalArea(shapes ...Shape) float64 { 
    var area float64
    for _, s := range shapes { 
        area += s.area()
    }
    return area
}

func main() {
    c := &Circle{5}
    r := &Rectangle{5, 10}
    t := totalArea(c, r)
    fmt.Println(t)
}

Interfaces helped us to use c and r as one type for totalArea.
